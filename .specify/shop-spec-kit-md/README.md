# Shop Feature – Shopify Integration (Handpans & Accessories)

This zip contains a GitHub Spec Kit-style set of Markdown docs for implementing a **Shop** section integrated with **Shopify Storefront API**, including:

- Main menu navigation link (feature-toggled)
- Product grid (image, title, short description, price, category)
- Minimal search + dropdown filters (category / price / availability)
- **No cart/checkout** — “Buy” links out to Shopify product pages
- Full Shopify setup guide (create store, add products, enable Storefront API, generate tokens)
- Configuration + deployment notes
